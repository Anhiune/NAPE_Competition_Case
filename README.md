# NAPE 2026 Energy Innovation Case Competition
## Strategic Investment Analysis for IPP Data Center Growth

This repository contains the full strategic analysis, financial modeling, and presentation materials for the 2026 SMU/NAPE Case Competition.

### Directory Structure

- **Analysis/**: Core strategic documents, detailed calculations, and executive summaries.
  - `DEEP_CASE_ANALYSIS.md`: Primary integrated analysis document.
  - `DETAILED_CALCULATIONS_ALL_TARGETS.md`: Step-by-step financial derivations.
  - `EXECUTIVE_SUMMARY_NAPE_CASE_ANALYSIS.md`: High-level summary of findings.
  - `SOURCES.md`: 49-source bibliography.
  - `create_slides.py`: Python script to generate the final presentation.
- **Data/**: Raw case materials, handbook, and text extractions.
  - `Case Data.xlsx`, `2026 SMU NAPE Case Final PDF.pdf`, `Round 1 Handbook .pdf`.
  - `files.zip`, `eia8602024.zip`: Raw data archives.
- **Output/**: Computed results, financial projections, and visual assets.
  - `NAPE_Round1_Presentation.pptx`: Final presentation file.
  - `*.json`, `*.csv`: Model outputs and scenario definitions.
  - `*.png`: Visual charts and dashboards.
- **EIA_Data_2024/**: Unzipped EIA-860 data source for market analysis.

### Getting Started

1. Review the [Executive Summary](Analysis/EXECUTIVE_SUMMARY_NAPE_CASE_ANALYSIS.md).
2. Explore the [Deep Case Analysis](Analysis/DEEP_CASE_ANALYSIS.md) for the strategic recommendation.
3. View the [Calculations](Analysis/DETAILED_CALCULATIONS_ALL_TARGETS.md) for financial validation.
